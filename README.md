# GERENCIADOR DE ALUNOS PARA ESCOLA

<p> Projeto de CRUD completo com Java Web </p>
 <p>Ferramentas utilizadas</p>
        <ul>
            <li>Spring</li>
           <li>Hibernate/JPA ( ORM )</li>
           <li>HTML 5</li>
  <li>CSS</li>
  <li>Javascript</li>
  <li> Bootstrap </li>
  <li> Thymeleaf </li>
  <li>H2 Database </li>
        </ul>
